　　本文来自佳友在线　　早年间佳能G1 X凭借着大尺寸（1.5英寸）CMOS传感器与便捷的机身操控性能，让当时的众多数码相机甚至是一部分微单/单电相机感到了前所未有的压力，而G1 X的后续机G1 X Mark II的表现也十分出色。如今佳能G系列相机已经发展壮大，G1 X Mark III也如期来到了我们的眼前。　　由于G1 X Mark III的定位是旗舰级DC，我们对其进行了详细的评测，此次客观评测带来的是关于相机客观表现的评测，而更多的功能应用以及样片表现将在后续的文章评测中为大家揭晓。　　此次佳能G1 X Mark III带来了更大的传感器，使用了与EOS 80D相同的有效像素约2420万的APS-C画幅大尺寸高感光度CMOS传感器，再一次重新定义了卡片相机画质的限界。　　轻量小巧的外形，APS-C画幅大尺寸传感器，使得该机被一些人亲切的称为称为口袋里的80D。那么这样一台倍受关注的相机实际表现又如何呢，恰逢G1 X Mark III来到佳友在线，我们在第一时间对这台相机进行了评测。　　　　虽然第一眼看上去，G1 X Mark III比其他G系列相机在外形上要突出了一些，取景器部位似乎也显得比较突出，但是实际上该机比前代产品G1 X Mark II要小巧，尤其是镜头收起时的长度只有上一代的一半左右。　　机身比前代小巧的原因之一，是机身镜头等效焦距从上代24-120mm变成了约24-72mm。虽然有所减少，但是作为卡片机来说这样的等效焦距也足够使用。　　G1 X Mark III外形极具操控感，按键布局合理，握持舒适。内置的电子取景器可实现类似单反相机的拍摄方式，得益于便捷的触摸与拖拽对焦功能，在使用电子取景器拍摄时的操作性也不逊于单反相机。　　比起前代只能上下翻转的180度翻转屏，此次G1 X Mark III还配备了支持触控的旋转式翻转液晶屏，可轻松应对非常规角度的拍摄和自拍等个性构图的需求。
