　　曾几何时，佳能G系列作为专业级卡片机的标杆，全世界各大旅游景区都可以见到它的身影。不过随着可换镜大底片无反相机的崛起，佳能的G系列逐渐失去了既有的光辉；而随着一众高端1英寸卡片机的出现，佳能G系列原有的操控、体积优势也不再明显。　　直到佳能 Powershot G1 X Mark III的出现，专业级卡片再一次的在画质、对焦、体积、操控等各个方面打了个完美的翻身仗。似乎从佳能 Powershot G1 X Mark III身上，我们看到了专业卡片机的新标准。　　优点：　　1、升级APS-C画幅传感器，画质提升明显；　　2、镜头综合素质不错，体积控制的也很好；　　3、全像素双核CMOS对焦表现优秀，触控拖动对焦体验一流；　　遗憾：　　长焦端光圈略小（F5.6）　　评测总结：坦率的说，我对前两代G1 X并不感冒。增大感光元件，的确能够带来更好的画质，不过这对于提升产品的用户体验作用不大。而到了佳能G1 X Mark III，画质再次攀升的同时，无论是电子取景器的加入，还是使用取景器时触屏拖动对焦，或是全像素双核CMOS的加入，都给了消费者实打实的体验升级。我们相信，这款产品一定会是佳能DC的一个里程碑式的产品。　　“终于，佳能G1 X Mark III内置了电子取景器”，这是我们不少媒体同行共同的感慨。讲真笔者在第一次见到这台机器的外观时，就几乎确定，这才是我们需要的专业级DC的样子。类似EOS M5、Powershot G5X的造型，支持电子取景器+翻转触控屏，佳能G1 X Mark III简直就是一台“小单反”。究竟这台相机在细节上有何心意，用料做工又表现如何，我们边看边聊。　　▲机身右侧的接口分别是遥控器接口、Micro USB数据接口（支持充电）、和Mini HDMI视频输出接口　　▲佳能PowerShot G1 X Mark III 取景器，硕大明亮、刷新率高无拖影，整体体验很好。　　总结：　　从外观设计来看，佳能G1 X Mark III确实是一台“精致的小单反”，甚至有些是单反所不具有的，比如说使用取景器时拖拽屏幕取景。而从做工来看，佳能G1 X Mark III也确实对得起旗舰DC的定位。金属的机身加上厚实的蒙皮，手感满满，让人觉着这台小机器很可靠。　　相对于佳能G1 X Mark III的高感表现，我们显然更关心它镜头的表现。毕竟，APS-C画幅的传感器加上 DIGIC 7 的处理器，通常高感画质都很好（参考EOS系列单反、EOS III配备了一块2420万像素的APS-C画幅传感器，运算使用佳能最新的 DIGIC 7处理器。这块“单反级”，哦不，其实就是佳能EOS 77D、EOS 800D单反上面的那块传感器，能给DC带来怎样的变化呢？我们还是通过实拍来看。　　我们把相机置于三脚架上，光圈F8，关闭一切机内降噪优化，直出JPEG文件。取各档ISO下图红框所示位置的100%放大截图。　　可以看到，佳能G1 X Mark III在ISO1600之前都能够表现出非常好的画质水平，ISO3200画面细节有所下降，整体仍比较干净。从ISO6400开始画面细节明显下降，并且出现了人眼可见的伪色和噪点。这种水平的控噪，与我们预计的大致相同，基本等同于佳能EOS系列 APS-C画幅单反。　　佳能佳能G1 X Mark III感光度使用建议　　鉴于我们的测试结果，我们建议日常使用佳能G1 X Mark III时，把感光度设定为ISO3200及以下自动即可。 　　一、对焦速度快——全像素双核CMOS AF带来的改变　　佳能G1 X Mark III是佳能小型数码相机中首次搭载全像素双核对焦的机型，CMOS上的所有像素都可以进行相差检测，因此并没有其他通过反差进行对焦的相机上面常见的“拉风箱”的现象。导致对焦一是快，二是一步到位，从不拖泥带水。　　二、对焦操作爽——拖拽对焦带来的体验上的升级　　佳能G1 X Mark III的屏幕支持使用取景器的同时，通过拖拽移动对焦点。笔者认为，有了这个功能，传统专业单反上的对焦摇杆都要逊色三分。一个是快速，指哪里对焦点就在哪里，不用一下一下的拨动摇杆来选择对焦点；二是很直觉，即便是小白用户也可以瞬间上手，学习几乎零成本；这个功能，目前看来是一个非常酷且又非常实用的“黑科技”，希望佳能可以尽快在全线产品中推广。　　最后老规矩，样张时间：
