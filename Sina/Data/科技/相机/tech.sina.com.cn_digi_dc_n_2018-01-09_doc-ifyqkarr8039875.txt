　　新浪数码讯 1月9日上午消息，松下在CES上推出新款微型单电数码相机LUMIX GH5S，视频记录功能进一步扩展，并在低光照条件下拥有更好的图像质量。这款为专业电影级视频工作者设计和开发的GH5S，实现了LUMIX相机历史上最高的ISO灵敏度和图像质量。新机售价为2499美元，折合16243元人民币。　　GH5S应用了双原生ISO技术和维纳斯引擎的全新1020万像素Live MOS传感器， 可实现最高ISO 51200的高灵敏度记录，再现图像中的暗部细节。维纳斯引擎的全新1020万像素Live MOS传感器可以在4：3，17：9，16：9和3：2等不同画幅比中在相同焦距下实现相同的视角。该传感器还支持14位RAW格式的照片拍摄，为专业RAW格式的工作流程开发提供更高的灵活性。　　LUMIX GH5作为无反相机（4K 60p / 50p（用于微型单电相机），4：2：2 10位（用于微型单电相机））在全球首次实现了4K 60p / 50p视频录制， 新的LUMIX GH5S再次在Cinema 4K（4096x2160）（拥有符合Digital Cinema Initiatives（DCI）定义的Cinema 4K（C4K：4096x2160）分辨率）规格上实现全球首个4K 60p / 50p视频录制 。它也可以进行4：2：2 10bit的内部视频录制（可以在SD存储卡上记录4：2：0 8bit的C4K 60p / 50p和4K 60p / 50p。），这是电影制作常用的颜色子采样规格，可忠实的进行色彩再现。LUMIX GH5S还可以在Cinema 4K 24p和4K 30p / 25p / 24p下录制4：2：2 10bit 400-Mbps All-Intra，以及在全高清格式下录制200-Mbps All-Intra。 而且4K、全高清视频录制都没有时间限制（直到SD存储卡已满或电池电量耗尽）。 LUMIX GH5S在照片风格下还拥有混合对数伽玛（HLG）4K HDR视频功能。 另外还提供适用于HLG的低比特率记录模式4K HEVC，可以在兼容HEVC压缩格式的AV设备（如Panasonic 4K HDR TV）上播放。　　VFR（可变帧率）可让用户以C4K / 4K（60 fps，最大速度减慢2.5倍，24p）和FHD（240 fps，最大速度减慢10倍，24p）录制升格/降格视频。 另外，GH5S相机中已预先安装了V-LogL，用户无需再单独购买软件升级密钥。 LUMIX GH5S兼容时间码IN / OUT。 通过闪光同步终端和BNC转接线及BNC线缆，设置时间码同步，很容易对多台摄像机拍摄的影片进行非线性编辑。 LINE输入也支持3.5mm麦克风插孔。　　DFD（散焦测距）技术（反差式自动对焦与DFD技术只适用于松下M4/3镜头）和超高速数字信号处理能力实现了0.07秒（AFS，使用H-ES12060（CIPA））的高速自动对焦。并在12位RAW时高速连拍可以达到12（AFS）/ 8（AFC）fps，14位RAW时可以达到11 （AFS） / 7 （AFC） fps。 225个对焦区域，追踪自动对焦，单点自动对焦和精确自动对焦等自动对焦功能可用于完成精确对焦。 4K PHOTO能够以大约800万像素的分辨率实现60fps的高速拍摄。　　作为一款在弱光拍摄方面表现出色的相机，LUMIX GH5S凭借其更高的灵敏度和优化后的传感器，具备低光照AF的-5EV的亮度检测性能。 Live View Boost是另一个实用功能，即使在完全黑暗的情况下也可以通过提高敏感度来实时观看画面。 夜间模式以红色显示菜单界面，在黑暗的情况下可以保护眼睛不受刺激。 MF辅助的放大倍率从以前10倍增加到20倍，这对天文摄影尤其便利。　　为了使GH5S足够坚韧，可以承受各种压力，它由镁合金全压铸的前/后框架组成，不仅防溅，防尘，而且可以防冻至零下10摄氏度。 GH5S配备了双SD卡槽，兼容高速，大容量的UHS-II和视频速度等级60的记忆卡。用户可以灵活选择接替录制，备份录制或分配录制等录制方法。 并提供HDMI Type A端子。　　GH5S有一个大型LVF（实时取景器），具有惊人的高倍率，约为1.52x / 0.76x（相当于35mm相机），提供120 fps的流畅显示速度。 高精度，高速OLED（有机发光二极管）显示屏具有3，680K点分辨率和100%的视野。 除了双拨盘之外，全向操纵杆还可以实现更直观，更灵活的操作。　　GH5S包括蓝牙和Wi-Fi?连接，提供更加灵活的拍摄体验和即时图像共享，操作简单。 与蓝牙4.2（称为BLE：低功耗蓝牙）兼容，能够以最小的功耗实现与智能手机/平板电脑的持续连接。 对于Wi-Fi，除了常规的2.4GHz（IEEE 802.11b / g / n）之外，还可以选择5GHz（IEEE802.11ac），以实现更安全稳定的连接。　　为延长电池寿命和稳定性，可使用电池手柄DMW-BGGH5（另售）。 XLR麦克风适配器DMW-XLR1（另售）允许使用外部XLR麦克风进行高分辨率录音（仅在MOV中）。
